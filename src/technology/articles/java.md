# Java 开发指南

Java 是一种广泛使用的面向对象编程语言，具有"一次编写，到处运行"的特性。

## Java 简介

- 面向对象
- 跨平台
- 安全可靠
- 性能优异

## 快速开始

### 安装 JDK

下载并安装最新版本的 JDK。

### Hello World

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

- 面向对象编程
