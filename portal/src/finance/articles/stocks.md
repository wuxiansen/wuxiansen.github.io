# 股票投资入门

股票投资是一种常见的投资方式，通过购买上市公司的股票，分享公司成长收益。本文介绍股票投资的基础知识。

## 什么是股票

股票是股份公司发行的所有权凭证，代表持有者对公司的所有权份额。

### 股票类型

- **A 股**：人民币普通股票
- **B 股**：人民币特种股票
- **H 股**：香港上市的内地公司股票
- **港股**：香港交易所上市股票
- **美股**：美国交易所上市股票

## 股票投资基础

### 如何开户

1. 选择证券公司
2. 准备身份证和银行卡
3. 下载开户 APP
4. 填写资料并视频认证
5. 绑定银行卡

### 交易规则

#### A 股交易规则

- **交易时间**：工作日 9:30-11:30，13:00-15:00
- **涨跌幅限制**：主板 ±10%，科创板/创业板 ±20%
- **交易单位**：100 股（1 手）为最小单位
- **T+1 制度**：当天买入，次日才能卖出

### 基本术语

- **开盘价**：当日第一笔交易价格
- **收盘价**：当日最后一笔交易价格
- **涨停**：当日最高涨幅
- **跌停**：当日最大跌幅
- **成交量**：交易的股票数量
- **市盈率（PE）**：股价/每股收益
- **市净率（PB）**：股价/每股净资产

## 投资策略

### 价值投资

- 寻找被低估的优质公司
- 长期持有
- 注重基本面分析
- 代表人物：巴菲特

### 成长投资

- 投资高成长性公司
- 关注行业趋势
- 重视公司发展潜力

### 趋势投资

- 跟随市场趋势
- 技术分析为主
- 注重买卖时机

## 风险管理

### 风险类型

1. **市场风险**：整体市场波动
2. **个股风险**：公司经营风险
3. **流动性风险**：无法及时买卖
4. **政策风险**：政策变化影响

### 风险控制

- **分散投资**：不要把鸡蛋放在一个篮子里
- **设置止损**：控制单次损失
- **仓位管理**：合理配置资金
- **理性投资**：避免情绪化决策

## 基本面分析

### 财务指标

- **营收增长率**：公司盈利能力
- **净利润率**：盈利质量
- **ROE（净资产收益率）**：资本使用效率
- **负债率**：财务安全性

### 行业分析

- 行业发展趋势
- 市场竞争格局
- 政策环境
- 技术创新

## 技术分析

### 常用指标

- **K 线图**：价格走势
- **均线**：趋势判断
- **MACD**：动量指标
- **KDJ**：超买超卖指标
- **成交量**：市场活跃度

## 投资建议

### 新手入门

1. **学习基础知识**：了解市场规则
2. **小额试水**：用少量资金练习
3. **建立投资体系**：形成自己的策略
4. **控制风险**：严格止损
5. **持续学习**：跟踪市场变化

### 长期投资

- 选择优质公司
- 分批建仓
- 长期持有
- 定期复盘
